canal-utils
=====================================

# 简介
本项目是基于阿里同步项目 [canal](https://github.com/alibaba/canal) 的一个工具箱帮助类库。

# 功能
## 转化器
Mysql DDL 语句转化到不同数据库上，支持：db2, hive, mssql, oracle, postgresql. [wiki 文档](http://www.baidu.com)
